== Description ==

OS-WPカスタマイズプラグインは、次のような機能があります。

= (1)管理画面のメニュー及び管理画面ページアクセス制限機能 =
* 権限グループ別にアクセスできる管理画面のページを指定できます。

= (2)権限グループの追加、編集機能 =
* 権限グループを新規追加、削除、利用できる権限の編集ができます。

= (3)ユーザ別、権限グループ別によるページ表示制限機能 =
* 記事内やウィジェットにショートコードを設置することによって、ユーザ別や権限グループごとにページ表示・非表示を切り替えられます。

= (4)独自ログインフォーム機能 =
* オリジナルのログインフォームを任意の記事内やウィジェット内に表示し、ログイン処理ができます。

= (5)その他のカスタマイズ。 =
* (5-1)記事の自動改行解除
* (5-2)アイキャッチ機能有効
* (5-3)HTML5マークアップ有効
* (5-4)タイトルタグの自動出力の有効
* (5-5)RSSフィードのリンク有効
* (5-6)ウィジェット内ショートコード有効
* (5-7)ナビゲーションメニューを有効
* (5-8)WPバージョン情報の非表示
* (5-9)管理画面内のバージョンアップ通知の非表示
* (5-10)管理画面フッターのWP文章の変更機能

これ以外にも、様々な機能を追加予定です。

== Installation ==

1. `/wp-content/plugins/`ディレクトリにアップロードします。
   Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. メニューの'プラグイン'で一覧を表示し、プラグインを有効化します。
   Activate the plugin through the ‘Plugins’ menu in WordPress
